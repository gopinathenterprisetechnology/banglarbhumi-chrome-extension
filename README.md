# Banglarbhumi Search Paper Generator

বাংলার ভূমি স্টাইলের সার্চিং পেপার তৈরি এবং পিডিএফ ডাউনলোড করার ক্রোম এক্সটেনশন।

## কিভাবে ব্যবহার করবেন (Installation Guide):
১. এই পেজের ডানদিকের উপরে **Code** বাটনে ক্লিক করে **Download ZIP** সিলেক্ট করুন।
২. জিপ ফাইলটি ডাউনলোড করে আপনার কম্পিউটারে আনজিপ (Unzip) করুন।
৩. Google Chrome ব্রাউজার খুলে অ্যাড্রেস বারে টাইপ করুন: `chrome://extensions/`
৪. ডানদিকের উপরে থাকা **Developer mode** অপশনটি অন (On) করুন।
৫. বামদিকের উপরে থাকা **Load unpacked** বাটনে ক্লিক করে আনজিপ করা ফোল্ডারটি সিলেক্ট করুন।
